<div class="s-prose js-post-body" itemprop="text">
<p>I'm parsing <code>HTML</code> with BeautifulSoup. At the end, I would like to obtain the <code>body</code> contents, but without the <code>body</code> tags. But BeautifulSoup adds <code>html</code>, <code>head</code>, and <code>body</code> tags. I <a href="https://groups.google.com/forum/#!topic/beautifulsoup/-VQdp2p0I8E" rel="noreferrer">this googlegrops discussion</a> one possible solution is proposed:</p>
<pre><code>&gt;&gt;&gt; from bs4 import BeautifulSoup as Soup
&gt;&gt;&gt; soup = Soup('&lt;p&gt;Some paragraph&lt;/p&gt;')
&gt;&gt;&gt; soup.body.hidden = True
&gt;&gt;&gt; soup.body.prettify()
u' &lt;p&gt;\n  Some paragraph\n &lt;/p&gt;'
</code></pre>
<p>This solution is a hack. There should be a better and obvious way to do it.</p>

<p>Do you mean getting everything inbetween the body tags?</p>
<p>In this case you can use :</p>
<pre><code>import urllib2
from bs4 import BeautifulSoup
page = urllib2.urlopen('some_site').read()
soup = BeautifulSoup(page)
body = soup.find('body')
the_contents_of_body_without_body_tags = body.findChildren(recursive=False)
</code></pre>

<div class="s-prose js-post-body" itemprop="text">
<p>I've found the easiest way to get just the contents of the body is to <code>unwrap()</code> your contents from inside the body tags.</p>
<pre><code>&gt;&gt;&gt; html = "&lt;p&gt;Hello World&lt;/p&gt;"
&gt;&gt;&gt; soup = BeautifulSoup(html, "html5lib")
&gt;&gt;&gt; print(soup)
&lt;html&gt;&lt;head&gt;&lt;/head&gt;&lt;body&gt;&lt;p&gt;Hello World&lt;/p&gt;&lt;/body&gt;&lt;/html&gt;
&gt;&gt;&gt;
&gt;&gt;&gt; soup.html.unwrap()
&lt;html&gt;&lt;/html&gt;
&gt;&gt;&gt;
&gt;&gt;&gt; print(soup)
&lt;head&gt;&lt;/head&gt;&lt;body&gt;&lt;p&gt;Hello World&lt;/p&gt;&lt;/body&gt;
&gt;&gt;&gt;
&gt;&gt;&gt; soup.head.unwrap()
&lt;head&gt;&lt;/head&gt;
&gt;&gt;&gt;
&gt;&gt;&gt; print(soup)
&lt;body&gt;&lt;p&gt;Hello World&lt;/p&gt;&lt;/body&gt;
&gt;&gt;&gt;
&gt;&gt;&gt; soup.body.unwrap()
&lt;body&gt;&lt;/body&gt;
&gt;&gt;&gt;
&gt;&gt;&gt; print(soup)
&lt;p&gt;Hello World&lt;/p&gt;
</code></pre>
<p>To be more efficient and reusable you could put those undesirable elements in a list and loop through them...</p>
<pre><code>&gt;&gt;&gt; def get_body_contents(html):
...  soup = BeautifulSoup(html, "html5lib")
...  for attr in ['head','html','body']:
...    if hasattr(soup, attr):
...      getattr(soup, attr).unwrap()
...  return soup
&gt;&gt;&gt;
&gt;&gt;&gt; html = "&lt;p&gt;Hello World&lt;/p&gt;"
&gt;&gt;&gt; print(get_body_contents(html))
&lt;p&gt;Hello World&lt;/p&gt;
</code></pre>
